# Todo App

## Problem Description

Build a small todo-list application **in Kotlin**. It needs to support two operations:

- An operation called **addTask** that accepts a string title and adds a new task to the list
- An operation called **markComplete** that accepts an integer ID and marks the corresponding task as complete

## Output Specification

Write the implementation to `output/TodoApp.<ext>`. Implement the two operations so they behave as described.
